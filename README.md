### ShareNowBackend


Share your invitations for events with other people.


#### Frontend Package

https://github.com/alinvelea27/ShareIt


#### Postman collection

Import file `./ShareNow.postman_collection` in Postman

#### How to run

***Backend***

1.) Import in VS Basic
2.) Run `dotnet run`
3.) Starts on port `5073`

***Frontend***

1.) Import in VS Code
2.) Run `npm run start`
3.) Starts on port `3000`
